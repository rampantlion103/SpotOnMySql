Business Objective:
Ability to process two files, one from ABS-CBN and another from Kantar to identify which of the spots differ from both files

Format Requirements:
The following worksheets will be used for comparison, Kantar:Spots = ABS-CBN:Detail

Filtering Date Instructions:
1. Enter the Start Date with the format MM/DD/YYYY to filter Kantar Spots Data
1. Enter the End Date with the format MM/DD/YYYY to filter Kantar Spots Data

Upload Instructions:
1. Upload the Kantar Spots Data on "Upload File 1" by clicking the "Browse..." button
2. Upload the ABS-CBN Report on "Upload File 2" by clicking the "Browse..." button
3. To clear or to update the file, click either the "Remove" or "Refresh" button

Download Instructions:
1. Press "Submit for Process" to begin identifying the spot differences
2. The spots reports will be automatically downloaded in a zip file which contains the MatchedSpots and UnmatchedSpots reports; this might take a few seconds or more
3. Whenever file formats are not compatible, error might occur; please check the Error Message which file is incorrect and repeat the submission
